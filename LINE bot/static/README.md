# LINE LIFF

การทำงานของ LINE LIFF
1. กด LINE LIFF แล้วลิงค์ไปยังหน้าเว็บไซต์สำหรับเลือกวันที่ แสดงเว็บไซต์แบบ tall พื้นหลังหน้าเว็บไซต์เป็นภาพเคลื่อนไหว มีช่องข้อความให้เลือกวันที่
และปุ่ม Ready
2. กดช่องข้อความให้เลือกวันที่ แล้วแสดงปฏิทินแบบ date picker กดเพื่อเลือก วัน/เดือน/ปี ได้
3. กดปุ่ม Ready แล้วมีการพิมพ์ text message เป็น "<วัน/เดือน/ปี ที่เลือก>" หลังกด ปุ่ม Ready เปลี่ยนเป็นปุ่ม Sent ทำหน้าที่เหมือนปุ่ม Ready
